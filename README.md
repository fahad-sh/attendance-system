# AttendPro — AI-Powered Attendance Management System

A full-stack office attendance system with face recognition, built on AWS EC2 with Docker.

## Live Demo
https://attendance-system-bumthanks59-1250s-projects.vercel.app

## Features
- Face recognition check-in using MediaPipe
- Admin dashboard with real-time tracking
- Automatic photo capture on attendance
- Leave management with approval workflow
- CSV export for payroll reporting
- IST timezone support

## Tech Stack
| Layer | Technology |
|---|---|
| Frontend | React 18 + Vercel |
| Backend | FastAPI + Python 3.11 |
| Database | MongoDB |
| Face Detection | MediaPipe |
| Infrastructure | AWS EC2 + Docker Compose |
| CI/CD | GitHub Actions |
| Security | JWT + bcrypt + Cloudflare |

## Cost
$0–$5/month on AWS Free Tier
